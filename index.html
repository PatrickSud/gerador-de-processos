1. Bem-vindo ao Gerador

1.1 O que é esta ferramenta? ▼

Este arquivo atua como um CMS Local (Sistema de Gerenciamento de Conteúdo). Ele permite que você crie documentações e manuais interativos diretamente no navegador, sem precisar de servidores ou bancos de dados.

  • Tudo o que você edita é salvo dentro do próprio arquivo HTML.
  • Você pode distribuir este arquivo final para qualquer pessoa, e ela terá a experiência de leitura interativa.
1.2 Como ativar o Modo de Edição? ▼

Para desbloquear as ferramentas de criação, siga estes passos:

  1. Clique no botão Editar Projeto no topo da tela.
  2. Se for um projeto existente, insira suas credenciais. Se for um projeto novo (como este), o modo de edição abrirá imediatamente.
  3. Ao exportar o projeto pela primeira vez, o sistema pedirá para você definir um usuário e senha exclusivos.
Dica de Acesso: Marque a opção "Lembrar credenciais" no login para pular a senha nas próximas edições da mesma máquina!

2. Criando seu Conteúdo

2.1 Adicionando e organizando seções ▼

Com o Modo de Edição ativo, você tem total controle sobre a estrutura do manual:

  • Role até o final da página e clique em + Adicionar Nova Seção para criar uma aba totalmente nova.
  • No menu lateral esquerdo, clique e arraste as seções para reordenar a exibição. O sistema ajustará tudo automaticamente.
  • Clique diretamente sobre o título de qualquer seção ou tópico para renomeá-lo.
  • O sistema conta com Histórico Inteligente: use Ctrl+Z para desfazer alterações e Ctrl+Y para refazer.
2.2 Formatando textos e mídias ▼

Utilize a barra flutuante no topo da tela para formatar seus textos:

  • Selecione o texto e aplique Negrito, Itálico, Sublinhado ou mude as cores e alinhamento.
  • Para inserir imagens ou GIFs, clique em + Adicionar Imagem ou YouTube no final do bloco. O sistema permite colar uma URL direta ou fazer upload de um arquivo do seu computador (que será salvo junto com o projeto!).
  • Para adicionar vídeos, use o mesmo botão e cole o link do YouTube.
  • Para criar caixas de dicas (como a abaixo), clique no botão 💡 (Anotações) na barra superior.

3. Salvando e Distribuindo

3.1 Como exportar o manual pronto? ▼

Após criar ou alterar todo o seu conteúdo, o passo final é compilar a ferramenta:

  • Clique no botão Salvar na barra inferior para registrar o progresso na memória.
  • Clique no botão laranja Exportar (.html).
  • Um arquivo será baixado para o seu computador. Esse arquivo já contém todas as imagens embutidas e os textos salvos, pronto para ser enviado para sua equipe!
Atenção: Sempre salve seu progresso antes de recarregar a página, ou as edições não salvas serão perdidas!
3.2 Como funciona a experiência de leitura do usuário final? ▼

O manual exportado funciona como um aplicativo interativo para quem o lê, oferecendo uma experiência focada e gamificada:

  • Acordeão Inteligente: Ao abrir um tópico, o anterior é fechado e marcado como lido automaticamente.
  • Trilha de Progresso: O menu lateral funciona como um stepper, mudando de cor (amarelo para em progresso, verde para concluído) conforme a leitura avança.
  • Busca Global: A barra de pesquisa filtra o conteúdo em tempo real em todas as abas.
  • Acessibilidade: O usuário pode alternar entre o tema claro e o modo escuro a qualquer momento.
Engajamento: A barra de progresso no topo motiva o leitor a concluir 100% da documentação!
